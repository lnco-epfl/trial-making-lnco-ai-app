# Trail Making Task - Instruction Texts

**Document for Neuropsychologist Review**

This document contains all instruction texts presented to participants during the Trail Making Task in English (EN) and French (FR). The Trail Making Task consists of two main cognitive subtasks designed to assess processing speed, visual scanning, and cognitive flexibility.

### Screen 2: Response Instructions & Error Handling

**Context:** Second instructional screen detailing specific response mechanics and what to do if an error occurs.

Click on each circle in the correct sequence order.

For numbers and letters: Click 1, A, 2, B, 3, C, etc.

**Important:** If you make a mistake: Click the last circle again to reset it, then continue with the correct sequence.

**FR:**

Pour les nombres seulement: Cliquez 1, 2, 3, etc.

### Practice 1 Interactive Trial

**Context:** Dynamic visual trial screen displaying 8 numbered circles randomly positioned on the screen. Participant clicks circles in ascending numerical order (1 → 2 → 3 → 4 → 5 → 6 → 7 → 8). A line is drawn between each successfully clicked circle. If an incorrect circle is clicked, the system requires the participant to click the last correct circle again to reset before continuing. The trial tracks:

- **Time taken**: Total duration from first click to completion
- **Errors (self-corrected)**: Incorrect clicks followed by correction
- **Errors (non-corrected)**: Incorrect clicks without immediate recovery

### Task 1 Interactive Trial

**Context:** Main task visual trial displaying 25 numbered circles (1-25) randomly distributed across the screen. Participant clicks circles in ascending numerical order. This task assesses:

- **Processing speed**: How quickly the participant can scan and identify numbers
- **Visual scanning**: Efficiency in locating target circles among many options
- **Motor speed**: Speed of clicking responses

Lines are drawn between correctly connected circles. If an incorrect circle is clicked, the system requires the participant to click the last correct circle again to reset before continuing.

**Configuration Parameters:**

- Number of circles: 25
- Circle radius: [CONFIGURABLE] pixels (default: 25)
- Sequence type: Numbers only (1-25)
- Layout: Randomly distributed positions (pseudorandomized to avoid systematic scanning patterns)
- Feedback during trial: Error detection (no success celebration to maintain focus)
- Performance tracking: Time taken, errors (self-corrected and non-corrected)

## EXPERIMENTAL DESIGN RATIONALE

The Trail Making Task is presented in a structured progression designed to assess two distinct cognitive domains:

### **Subtask A (Practice 1 & Task 1): Numbers Only**

- **Cognitive Demands**: Visual scanning, processing speed, motor speed
- **Baseline Measure**: Performance on simple, sequential numerical tracking
- **Clinical Relevance**: Detects slowing from various neurological conditions

### **Subtask B (Practice 2 & Task 2): Numbers and Letters**

- **Cognitive Demands**: Visual scanning, processing speed, motor speed, **PLUS cognitive flexibility**
- **Additional Load**: Alternating between two stimulus categories
- **Clinical Relevance**: Differentiates processing speed impairments from executive dysfunction

### **Error Tracking Methodology**

- **Self-Corrected Errors**: Indicates metacognitive awareness and self-monitoring ability
- **Non-Corrected Errors**: May indicate inattention or difficulty recognizing errors
- **Timing**: Captures how delay in error detection affects overall performance

### **Practice Phases**

Two practice sessions (simpler sequences: 8 circles, then 8 mixed circles) prepare participants for the main tasks while providing opportunity to understand the response mechanics and avoid learning effects during main task assessment.

## NEUROPSYCHOLOGICAL INTERPRETATION

The Trail Making Task is sensitive to:

1. **Frontal Lobe Dysfunction**: Particularly impaired performance on Task 2 (requires executive function)
2. **Subcortical Pathology**: Slowed processing speed affecting both subtasks
3. **Right Hemisphere Lesions**: Impaired visual scanning and spatial organization
4. **Dementia**: Disproportionate increase in errors (self-correction breakdown)
5. **ADHD/Concentration Issues**: Increased non-corrected errors despite correct speed
6. **Motor Slowing**: Slower time on all tasks without increased error rate

---

## QUALITY ASSURANCE FOR RESEARCHERS

When reviewing participant data:

- **Expected Practice Performance**: Novice participants typically take 15-30 seconds for Practice 1
- **Expected Task 1 Performance**: Adults typically complete in 25-40 seconds with 0-3 errors
- **Expected Task 2 Performance**: Adults typically complete in 40-70 seconds with 0-5 errors (increased difficulty)
- **Red Flags**: Any participant completing Task 2 in <20 seconds may indicate rushing/lack of effort
- **Screening**: Task 1 completion >90 seconds may indicate processing speed impairment or motor difficulties
